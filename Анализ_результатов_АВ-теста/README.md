# Приоритизация гипотез по увеличению выручки интернет-магазина и анализ результатов А/В-теста

## Источник данных:
* Внутренные данные интернет-магазина: файл с гипотезами и файл с заказами и посещениями сайта за период

## Задачи исследования:
* Определить приоритеты в проверке гипотез с помощью фрейморков ICE и RICE
* Провести исследовательский анализ данных о заказах и посещениях интернет-магазина
* Построить графики кумулятивных метрик (конверсия и средний чек) по группам участников А/В-эксперимента, произвести сравнение
* Осуществить расчет и сделать выводы и статистической значимости различий в метриках между экспериментальными группами, принять решение о целесообразности продолжения эксперимента

## Используемые инструменты и методы
python, pandas, numpy, matplotlib, skipy

## Навыки
A/B-тест, проверка статистических гипотез, фреймворки ICE и RICE

## Сфера деятельности
Интернет-магазины
