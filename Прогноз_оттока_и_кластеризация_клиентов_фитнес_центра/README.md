# Прогнозирование оттока и кластеризация клиентов сети фитнес-центров

## Цель исследования:
Провести анализ данных и подготовить рекомендации для сети фитнес-центров в рамках разработки стратегии взаимодействия с клиентами и их удержания.
    
## Задачи исследования:
* Спрогнозировать вероятность оттока (на уровне следующего месяца) для каждого клиента;
* Сформировать типичные портреты клиентов: выделить несколько наиболее ярких групп и охарактеризовать их основные свойства;
* Проанализировать основные признаки, наиболее сильно влияющие на отток;
* Сформулировать выводы и разработать рекомендации по повышению качества работы с клиентами:
    * выделить целевые группы клиентов;
    * предложить меры по снижению оттока;
    * определить другие особенности взаимодействия с клиентами.

## Исходные данные:
Сsv-файл с информацией о клиентах на основе журнала посещений, покупок, информацией о текущем статусе абонемента клиента, а также данные клиента за предыдущий до проверки факта оттока месяц.

## Используемые библиотеки
pandas, matplotlib, seaborn, scikit-learn, skipy

## Навыки
Машинное обучение, классификация, кластеризация
