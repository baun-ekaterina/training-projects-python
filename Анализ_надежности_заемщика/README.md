# Исследование надежности заемщиков коммерческого банка

## Источник данных:
* Внутренние данные о заемщиках коммерческого банка

## Задачи исследования:
* Осуществить предобработку исходных данных:
	* обогащение данных (заполнить пропуски)
	* проверка на аномалии
	* устранение дубликатов
* Сформулировать ответы на вопросы о влиянии различных клиентских характеристик на возврат кредита в срок

## Используемые инструменты и методы
python, pandas

## Навыки
Обработка данных, дубликаты, пропуски, категоризация, декомпозиция

## Сфера деятельности
Банковская сфера, Кредитование
